## Getting Started

# Live Site Demo:
[fuel-finder.live](https://fuel-finder.live)

# Developers' Guide:
First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see application.
